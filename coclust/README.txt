Installation
============

The package relies on the NumPy, SciPy and scikit-learn libraries, and also on
Matplotlib for some of the demos included in the package.

If these libraries are already installed on your machine, you can install
coclust from PyPI by just entering:

    pip install coclust

See https://pypi.python.org/pypi/coclust and http://coclust.readthedocs.org for
additional installation and usage notes.

The package can also be installed from the provided source code by running:

    python setup.py install
